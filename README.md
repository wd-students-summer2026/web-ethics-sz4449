# Web Ethics Exercise
You will research and present an ethical issue related to the Internet and/or The Web. The group will discuss and debate this issue during the class session.

## Issue overview

CAPTCHA and web accessibility is an ethical issue because many websites use human-verification tests to decide who can enter, submit a form, buy tickets, create an account, or log in. Common CAPTCHA tests ask users to type distorted text, select images, solve puzzles, or click “I’m not a robot.” Supporters argue that CAPTCHA protects websites from bots, spam, fake accounts, ticket scalping, credential attacks, and other automated abuse. However, critics argue that CAPTCHA can also block real human users, especially people with visual, hearing, cognitive, or motor disabilities. The ethical question is whether website security should depend on tests that may make the web less accessible for some people.

One side of the debate argues that CAPTCHA is necessary for website security. From this perspective, websites need some way to tell the difference between human users and automated bots. Without CAPTCHA or similar tools, online forms could be flooded with spam, accounts could be attacked by automated login attempts, and ticket or shopping websites could be dominated by bots. Supporters believe that a small amount of user inconvenience may be acceptable if it protects the website and other users.

The other side argues that CAPTCHA creates unfair barriers to web access. From this perspective, a security tool should not make basic website use difficult or impossible for some people. Image CAPTCHAs may be hard for blind or low-vision users, audio CAPTCHAs may be difficult for deaf or hard-of-hearing users, and confusing puzzles may create problems for users with cognitive disabilities. Critics also argue that CAPTCHA often frustrates ordinary users and may collect extra behavioral or device data in the background. They believe websites should use more accessible and privacy-friendly alternatives instead of forcing users to prove they are human through difficult tests.
